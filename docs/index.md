# kubeadapt-cli

Full documentation is available at [kubeadapt.io/docs/v1/cli/overview](https://www.kubeadapt.io/docs/v1/cli/overview/).

## Development

```bash
task build      # Build binary for current platform
task test       # Run tests with race detector
task lint       # Run linter
task fmt        # Format code
task vuln       # Vulnerability check
```
